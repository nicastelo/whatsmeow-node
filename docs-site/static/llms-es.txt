# whatsmeow-node

> WhatsApp para Node.js — powered by the most battle-tested WhatsApp Web library

whatsmeow-node es un cliente Node.js completamente tipado para WhatsApp Web. Encapsula [whatsmeow](https://github.com/tulir/whatsmeow), la biblioteca Go que alimenta el Mautrix WhatsApp bridge (usado por miles de usuarios de Matrix diariamente), y lo expone como un paquete npm nativo con 100 métodos tipados.

El binario Go se comunica con Node.js vía JSON-line IPC sobre stdin/stdout. Sin necesidad de compilación nativa — se proporcionan binarios precompilados para macOS, Linux y Windows (x64 y arm64).

## Características Principales

- Upstream probado en batalla: respaldado por whatsmeow, la biblioteca WhatsApp Web más confiable
- Ligero: binario precompilado de 10–20 MB, sin dependencia de Chromium o navegador
- Completamente tipado: 100 métodos asíncronos con interfaces TypeScript
- Multi-plataforma: macOS, Linux, Windows (x64, arm64, musl)
- Sesiones persistentes: SQLite (dev) o PostgreSQL (prod)

## Quick Start

```typescript
import { createClient } from '@whatsmeow-node/whatsmeow-node';

const client = createClient({ store: 'session.db' });
await client.init();
const qr = await client.getQRChannel();
await client.connect();
// Escanea el código QR, luego envía un mensaje:
await client.sendMessage('1234567890@s.whatsapp.net', { conversation: '¡Hola!' });
```
