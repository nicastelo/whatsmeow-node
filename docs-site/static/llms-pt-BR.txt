# whatsmeow-node

> WhatsApp para Node.js — powered by the most battle-tested WhatsApp Web library

whatsmeow-node é um cliente Node.js totalmente tipado para o WhatsApp Web. Ele encapsula o [whatsmeow](https://github.com/tulir/whatsmeow), a biblioteca Go que alimenta o Mautrix WhatsApp bridge (usado por milhares de usuários Matrix diariamente), e o expõe como um pacote npm nativo com 100 métodos tipados.

O binário Go se comunica com o Node.js via JSON-line IPC sobre stdin/stdout. Sem necessidade de compilação nativa — binários pré-compilados são fornecidos para macOS, Linux e Windows (x64 e arm64).

## Recursos Principais

- Upstream testado em batalha: backed by whatsmeow, a biblioteca WhatsApp Web mais confiável
- Leve: binário pré-compilado de 10–20 MB, sem dependência de Chromium ou navegador
- Totalmente tipado: 100 métodos assíncronos com interfaces TypeScript
- Multi-plataforma: macOS, Linux, Windows (x64, arm64, musl)
- Sessões persistentes: SQLite (dev) ou PostgreSQL (prod)

## Quick Start

```typescript
import { createClient } from '@whatsmeow-node/whatsmeow-node';

const client = createClient({ store: 'session.db' });
await client.init();
const qr = await client.getQRChannel();
await client.connect();
// Escaneie o QR code, depois envie uma mensagem:
await client.sendMessage('1234567890@s.whatsapp.net', { conversation: 'Olá!' });
```
