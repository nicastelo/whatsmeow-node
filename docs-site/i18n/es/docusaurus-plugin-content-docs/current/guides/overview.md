# Guías

Tutoriales prácticos que te guían a través de tareas comunes de automatización de WhatsApp con whatsmeow-node.

| Guía | Descripción |
|------|-------------|
| [Crear un Bot](build-a-bot) | Crea un bot de WhatsApp desde cero — recibe mensajes, maneja comandos, envía respuestas |
| [Enviar Stickers](send-stickers) | Sube y envía stickers WebP, descarga stickers entrantes |
| [Descargar Multimedia](download-media) | Guarda imágenes, videos, audio y documentos recibidos en disco |
| [Indicadores de Escritura](typing-indicators) | Muestra indicadores de "escribiendo..." y "grabando audio...", suscríbete a la presencia de otros |
| [Vincular WhatsApp](pair-whatsapp) | Vincula una cuenta de WhatsApp mediante QR code o código de número telefónico |
| [Conectar con IA](connect-to-ai) | Construye un chatbot de WhatsApp con IA usando Claude |

## ¿Buscas fragmentos de código?

La sección de [Ejemplos](/docs/examples/overview) tiene código enfocado y listo para copiar y pegar para funciones específicas — multimedia, grupos, encuestas, reacciones y más.
