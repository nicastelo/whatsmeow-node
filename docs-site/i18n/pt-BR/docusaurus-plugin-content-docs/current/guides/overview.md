# Guias

Tutoriais práticos que mostram como realizar tarefas comuns de automação no WhatsApp com whatsmeow-node.

| Guia | Descrição |
|------|-----------|
| [Criar um Bot](build-a-bot) | Crie um bot de WhatsApp do zero — receba mensagens, trate comandos, envie respostas |
| [Enviar Stickers](send-stickers) | Faça upload e envie stickers WebP, baixe stickers recebidos |
| [Baixar Mídias](download-media) | Salve imagens, vídeos, áudios e documentos recebidos no disco |
| [Indicadores de Digitação](typing-indicators) | Mostre indicadores "digitando..." e "gravando áudio...", inscreva-se na presença de outros |
| [Parear WhatsApp](pair-whatsapp) | Vincule uma conta WhatsApp via QR code ou código por número de telefone |
