# @whatsmeow-node platform binary

This package contains the precompiled [whatsmeow-node](https://github.com/nicastelo/whatsmeow-node) Go binary for a specific platform.

**You should not install this package directly.** Instead, install the main package:

```bash
npm install @whatsmeow-node/whatsmeow-node
```

The correct binary for your platform is selected automatically via `optionalDependencies`.
